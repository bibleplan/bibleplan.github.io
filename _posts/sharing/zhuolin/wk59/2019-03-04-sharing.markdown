18:16-19:2，迦密山  
以利亞的服事:  
+ 指摘亞哈和他父家的罪；  
+ 責備百姓的三心二意，激勵他們堅定心意跟隨神；  
+ 代表以色列十二個支派，獻祭禱告求雨蒙應允。自從以色列人進入迦南地，十二個支派從來沒有真正的合一，到了南北國分裂以後更是戰爭不斷，弟兄反目成仇。北國從耶羅波安開始一直試圖將南北敬拜分開。以利亞重修祭壇、代表十二個支派向神禱告的，表明耶和華是整個以色列的神。  
+ 除去巴力的眾先知450人；  

以利亞面臨的挑戰:  
+ 亞哈不認罪反而說以利亞是乾旱的罪魁禍首；  
+ 百姓持觀望態度，並不專心跟隨神；  
+ 孤身一人面對450巴力先知，沒有屬靈同伴；  

神所行的神蹟:  
+ 降火在完全濕透的祭壇，燒盡燔祭，連濕透的木柴、石頭、塵土都燒盡，溝裡的水都燒乾；  
+ 降雨；  
+ 神的靈降在以利亞身上，讓他徒步奔跑卻快過亞哈的車馬；  

結果:  
+ 亞哈從一開始對以利亞的指責，變為聽以利亞的話“上去吃喝”等待降雨，雨將下的時候也聽從以利亞的話“坐車往耶斯列去”；  
+ 百姓們俯伏在地承認耶和華是真神，並且聽從以利亞，殺掉巴力的先知，表示與巴力崇拜斷絕；  
+ 耶洗別並不悔改，反要殺害以利亞。亞哈對此卻完全沒有主意，好似主耶穌撒種的比喻中落在路邊的種子，沒有發芽就被飛鳥吞吃了；

19:3-21 何烈山（西乃山）  
但凡真正的先知，在社交上都很不討喜，因為他們的任務就是罵人或命令人。但是以利亞在社交上是更是格外得愚拙。他說話的方式只有兩樣，一種是對人，一種是對神，涇渭分明:面對人的時候，以利亞管他面對的是君王，還是百姓，是悖逆的人，還是敬虔的人（俄巴底），是順利的人，還在在苦難中的人（寡婦），他的話都是直接了當的命令式，從來不會看人下菜碟兒，也沒有對苦難的人（寡婦）的安慰，也沒有對敬虔的人（俄巴底）的鼓勵，看起來他絕對不合適做輔導；他面對神的時候，則是全然信靠、自卑、順服、透明敞開、沒有半點虛假。他透明真實得可愛。當巴力的眾先知獻祭禱告沒有效果的時候，他在一旁嘲笑貶損他們:你們倒是大聲求吿啊！再大聲點哪！蹦吧跳吧！再起勁兒點哪！快把你們的巴力給叫醒啊！怎麼啦？是不是他聽不見哪？！（“到了正午， 以利亞 嬉笑他們，說:「大聲求告吧！因為他是神，他或默想，或走到一邊，或行路，或睡覺，你們當叫醒他。」” 列王紀上 18:27）

在迦密山上，是他服事神生涯的大勝仗的時刻，他飛跑在亞哈的車馬前的時候，心中是怎樣的期盼和歡暢！明天等待他的是什麼呢？會不會是屬靈的大復興？也許明天開始，他就不必東躲西藏，而是可以挽起袖子開始各樣的事工。先從哪裡開始好呢？打碎巴力的邱壇，驅逐偶像崇拜？重建倒塌的祭壇，敬虔事奉神？門訓教導一批年輕的真正敬畏神的先知？催促亞哈與猶大國建立友好關係，使百姓可以去耶路撒冷敬拜？太多的事情要做，太多的事情可以做，真是心情激動，摩拳擦掌。——誰曉得他等到的卻是耶洗別的死亡通輯令。

以利亞的第一反應就是“起來逃命”，這跟以前他是跟著神的吩咐而行完全不同（回頭翻閱他每次換地方，都是神吩咐，他馬上照著行），這次他是懼怕、失望到了極點。

他坐在那裡求死（19:4），讓人想起另一位求死的先知，就是被太陽曬得頭昏眼花的約拿，不過約拿的“求死”更多是在抱怨，而以利亞的“求死”真真的是因事工的“失敗”而心灰意冷（“因為我不勝於我的列祖”）。

19:5-18這一段裡神對祂疲憊沮喪的僕人所行的事，實在是讓人心中溫暖而柔軟。正是“疲乏的，祂賜能力；軟弱的，祂加力量”（以賽亞書40:29）。5-8，神沒有直接回答以利亞的話，而是差派祂的天使給他帶來食物和水，叫醒他起來吃喝，天使應當是一直守護在他身邊，直到他休息充足，才第二次把他叫起來吃東西喝水。這食物是天上的食物，這水是天上的水，為以利亞加添心力，使他在曠野四十晝夜行路卻不疲倦，正如摩西在西奈山上四十天一樣。

19:8-9，以利亞在乾枯沮喪的時候，他想的是去神的山何烈山（就是西奈山），正如大衛在逃避掃羅的時候，第一站去的就是祭司那裡，他逃避押沙龍的時候，第一站去的是橄欖山禱告。我們若是在做主的工失望疲憊的時候，也同樣要單獨回到神那裡去（以利亞將他的僕人留在別是巴，不叫他跟隨）。

19:10-13，在那裡，神兩次問他“以利亞啊，你在這裡做什麼？”，顯然以利亞來此不是神叫他來的，而是他自己來的。以利亞對神的回答，是我們在心情低落的時候應當學習的榜樣:他描述自己的感受，闡述事實，而不是指責別人，他甚至連耶洗別這最該被罵的人都沒有罵！他對神的回答，顯見得在他與神之間沒有任何其他的人，如此敞開透明，如同單純的孩子。從他的回答，我們也能體會到前面神的詢問“你在這裡做什麼？”，絕對不是責問，而是溫柔地呼喚。神叫他出來，到神的面前。舊約裡唯有寥寥幾人這樣見過神，亞伯拉罕是一個，摩西是一個，以利亞也是一個。

“耶和華說:「你出來站在山上，在我面前。」那時耶和華從那裏經過，在他面前有烈風大作，崩山碎石，耶和華卻不在風中；風後地震，耶和華卻不在其中； 地震後有火，耶和華也不在火中；火後有微小的聲音。”（列王紀上 19:11-12）

耶和華不在崩山碎石的狂風中，不在天搖地動的地震中，也不在熊熊焚燒的烈火中，卻在這一切威猛之後的微小聲音中。以利亞自己也不是被風、地震和火的威勢帶領出來，卻是被神微小的聲音吸引出來（19:13）。我們的神的行事多麼可畏，又多麼奇妙。祂沒有趁著迦密山上的大勝利，緊鑼密鼓進行更多風、火和地震的事工，卻是要用微小的聲音去滲透到人的心中，因為百姓們也正如以利亞，吸引他們的不是神蹟，而是神的聲音。這也正是耶和華要以利亞去做的:用眾先知微小的聲音，繼續在百姓心裡做工，而神早已經為他預備同工，並且神掌管歷史的車輪，惡人必然被審判。以利亞離開西奈山，第一件事竟然就是遇到願意捨棄富足生活跟隨他的以利沙！從此以利亞在神交託的事上不再是孤軍奮戰。

神並不順著我們的心思，寵溺我們的情緒，但是祂所的供應正是我們真正的需要，正如主耶穌在世上行走的時候，總是關注、服事人真正的需要；神不見得直接回答我們的問題，而是使用我們周圍發生的事情來帶領我們明白；祂大多時候也不是用振聾發聵的聲音，而是聖靈用微小的聲音對著我們的靈說話。但是這些卻帶來屬靈的安息，景況大約沒有改變，但是靈裡卻重新得力！

`Zhuolin`

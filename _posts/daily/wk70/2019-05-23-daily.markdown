### 默想：神的故事
+ 末底改撕裂衣服，穿麻衣，蒙灰塵，痛哭哀號。

+ 末底改囑咐以斯帖進去見王，為本族的人在王面前懇切祈求。

+ 以斯帖要猶大人禁食三晝三夜，然後違例進去見王。若死就死吧！

+ 王見王后以斯帖站在院內，就施恩於她，向她伸出手中的金杖。

+ 哈曼立木架，以便殺害末底改。

### 默想：我的故事
+ 【服事忠誠】末底改要求以斯帖冒生命危險出頭援救。為何他沒有求神的幫助？神如何預備我們為今天所有棘手的經歷而思考的東西。

### 問題：神如何拯救希西家王？ 

{%- include BibleLinks.html -%}

### 默想：神的故事 
+ 希西家使家宰和書記去見先知以賽亞，以賽亞預言耶和華拯救。 

+ 耶和華的使者在亞述營中殺了十八萬五千人。 

+ 耶和華醫治希西家，加增他十五年的壽數。 

+ 希西家給巴比倫王的使者觀看寶庫的財寶。 

+ 以賽亞預言猶大被擄到巴比倫去。 

### 默想：我的故事
+ 【價值定位】希西家對神的禱告重點是什麽？我們可以學到禱告的哪些原則？

+ 【生命品格】為何希西家向巴比倫使者展示財富會帶來以賽亞的預言？我們應該如何對待神對我們的祝福？

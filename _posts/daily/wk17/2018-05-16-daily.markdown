### 問題：摩西有關訴訟的審判、立王有哪些吩咐？神通過摩西應許賜下怎樣的先知？

{% include BibleLinks.html %}

### 默想：神的故事 
+ `16:18-20` 和 `17:8-13` 講到祭司和審判官，強調審判官要公義地審判，不可屈枉正直，不可受賄；訴訟的百姓要聽從祭司和審判官。

+ `16:21–17:7` 和 `18:9-13` 吩咐不可設立偶像，不可獻有殘疾有病的牛羊，要懲治拜偶像之人，不可學異邦國民所行可憎的事。

+ `17:14-20` 講論神允許立王及王的條例。注意所羅門王在晚期洽洽有違這些王的條例。

+ `18:1-8` 講述祭司和利未人當得的分。

+ `18:10-22` 應許`「興起一位先知，像你。我要將當說的話傳給他；他要將我一切所吩咐的都傳給他們。誰不聽他奉我名所說的話，我必討誰的罪。」`指向救主耶穌。

### 默想：我的故事 
+ 【與神同行】主耶穌就是神所應許賜下的那位先知，若不接受祂的救恩，神在末日就必追討人的罪。接受耶穌救恩的人必願意遵從祂的教導，請省察自己對耶穌的教
導的態度，並以禱告回到神的身邊。

+ 【生命品格】神在這裡再次提到要嚴懲拜偶像的人。當今的世界裡其實有太多的偶像在取代著神的位置。請自我省察，看自己是否仍然在用工作、家人、財物、娛樂取代神而作為自己喜樂的來源？

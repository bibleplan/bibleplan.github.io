{% include BibleLinks.html %}

### 默想：神的故事 
+ 神為過失殺人者預備了可以存活的逃城，但故意殺人者就要被處死。

+ 無論何罪，都需要兩三個人作見證；不可作假見證。

+ 出去爭戰時不要懼怕，但新建了房屋、新種了葡萄園、剛聘了妻的人可以留下，膽怯的人也可以留下；攻打之前，要先宣告和睦的話，執意為敵的，則要毀滅。

### 默想：我的故事 
+ 【肢體關係】神設立逃城，因為知道世人會為他人對自己的傷害而報復，即使這些傷害是出於他人無心的舉動。我最近有沒有需要原諒他人對自己的傷害？

+ 【生命品格】神充分體諒新建了房屋、新種了葡萄園、剛聘了妻的人，讓他們不須馬上參與征戰。我可以如何設身處地為他人著想？

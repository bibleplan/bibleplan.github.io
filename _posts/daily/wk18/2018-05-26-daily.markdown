### 問題：摩西的詩歌寫了什麼？對十二支派有哪些祝福？

{% include BibleLinks.html %}

### 默想：神的故事 
+ `32 章`：按照神的吩咐，摩西寫下詩歌，回顧神的恩典與作為，百姓的悖逆不信，隨之而來的責罰，也帶著回歸復興的應許。詩歌宣告神是磐石，除祂以外，並無別神。摩西叮囑百姓要把這警教放在心上，要吩咐子孫謹守遵行，因為`「這不是虛空、與你們無關的事，乃是你們的生命」`。神再次告訴摩西，他因米利巴水的事上沒有尊神為聖而將死在尼波山上。

+ `33 章`：摩西臨死前向以色列十二支派祝福，以形容神的威榮開始，以讚美蒙神拯救之民所承受的恩典結束。以色列雖然必定失敗，在這祝福中看到神的憐憫，神的選民是何等有福的。

### 默想：我的故事 
+ 【生命品格】以色列民悖逆不信，拜那些對他們本無益的假神。我們也常常花太多精力追逐世上那些本無永恆意義的事和物，忘記積攢那天上有永恆價值的財寶，正像那些在美麗的海灘卻只知玩泥巴的孩童。請在神面前禱告，求神讓我們不再短視。

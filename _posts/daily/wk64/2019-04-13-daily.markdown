{%- include BibleLinks.html -%}

## 默想：神的故事
+ 所羅門在阿珥楠的禾場上開工建造耶和華的殿。

+ 所羅門建築神殿的根基都按著古時的尺寸。

+ 在至聖所按造像的法子造兩個基路伯，用金子包裹。

+ 在殿前造了兩根柱子，又製造一座銅壇，又鑄一個銅海。

+ 又製造十個盆，十個金燈臺。又建立祭司院和大院並院門。

+ 戶蘭又造了盆、鏟、碗，和各樣器具。

### 默想：我的故事
+ 【服事忠誠】所羅門建聖殿，所用器具、材料、方法都非常精緻。我們在為主服事的時候，是否也是盡心盡力，精益求精，完完全全彰顯神的恩賜的榮耀呢？

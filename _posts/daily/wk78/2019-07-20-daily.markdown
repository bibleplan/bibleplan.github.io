{%- include BibleLinks.html -%}

### 默想：神的故事
+ 耶和華必再揀選以色列，將他們安置在本地。

+ 耶和華讓人得享安息的日子，先知必題詩歌說，耶和華折斷了惡人的杖，轄制人的圭。

+ 非利士全地必以饑荒治死，你所餘剩的人必被殺戮。

+ 一夜之間，摩押的亞珥和摩押的基珥變為荒廢，歸於無有。


### 默想：我的故事
+ 【生命品格】驕傲的天使要升到天上，高舉寶座在神眾星以上。但因此他必墜落陰間，到坑中極深之處。驕傲是我們的惡的起首，今天我需要如何處理自己的驕傲？

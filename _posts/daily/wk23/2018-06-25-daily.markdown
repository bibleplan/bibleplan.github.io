### 本週引言

本週我們將結束士師記的讀經，並進入路得記。

在士師記中讓我們看到，除了以色列人的悖逆，還有在人的罪中神的一次次拯救。耶和華啊，求你拯救我脫離惡人的手，保護我脫離強暴的人圖謀推我跌倒。驕傲人為我暗設網羅和繩索，他們在路旁鋪下網，設下圈套。有時世界看起來好像邪惡控制，不法橫行，神仿佛不再聽禱告。不公正的統治和不道德的人管轄，不敬畏神如亞比米勒的人的天下。這是墮落世界難以避免的現實，不止在當年的以色列或今天的極權國家。當人們按著自己的私慾生活，國家和社會中不能容下耶和華。但人的邪惡絕不能否定神的主權，在動亂的環境中耶和華仍然為大，神允許事情發生，但結果是祂控制，公義終於會彰顯，邪惡不能逃脫刑罰。

### 默想：神的故事 
+ `「那時以色列中沒有王，各人任意而行。」(士17:6)`

+	米迦造神像和以弗得，在自家私設聖所，指派自己的兒子作祭司。

+ 從伯利恒來尋找供養的利未少年人，留下作米迦家中的祭司。

+	不在摩西約書亞分派的地界爭戰得地、卻四處尋找易得的居住地的但支派，看到米迦私設神堂中的偶像、以弗得就搶去，並利誘利未少年人轉去作但支派的祭司。

+ 但支派野蠻屠殺在拉億安居的民，奪了那地居住，改名為但，設立摩西之孫作但支派的祭司。

### 默想：我的故事 
+ 【生命品格】米迦和他母親雖然説是在敬拜耶和華，卻因為他們在按著自己的方法，已經於不知不覺中轉而在拜偶像。讓我們檢視自己，看我們是在盡心盡意盡力敬拜那獨一的真神，還是只是在自己方便的時候、用自己方便的方式敬拜？

+	【服事忠誠】那利未少年人不在神所指定的地點服事神，為著屬世的利益，先是作米迦一家的祭司，後又改作但支派的祭司，全然沒有尋求神的旨意。我們在服事的時候，是謙卑擺上自己、跟隨神的帶領，還是只做能給自己帶來好處的事工？

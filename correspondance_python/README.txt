These crude scripts do: 
1. find the correspondance between 2022-2023 cycle and 2020-2021 cycle based on manually-formatted full_list.xlsx, 
2. copy paste Zhuolin's notes and daily posts to the new cycle with the new date,
3. generate template for Bin's bible daily recording
4. find the correspondance between 2022-2023 cycle and 2018-2019 cycle
5. copy paste English version bible daily to the new cycle with the new date.

Note: In English version, there is no one-to-one correspondance from 2018 cycle to 2022 cycle. This may affect a couple of days post but effect should be minimal.
